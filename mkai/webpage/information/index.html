면까몰 AI 이용방법

1. 이력서를 준비하세요.

jpg, png 등의 이미지로 된 이력서를 준비해주세요.

AI가 자동으로 이력서를 분석해 경력사항을 분석해줍니다.

이력서가 없어도 스스로 채울 수 있어요.

2. ChatGPT API Key를 준비하세요.

AI를 사용하기 위해서는 ChatGPT의 API Key가 필요합니다.

[시작하기] 버튼을 누르면 API Key를 입력할 수 있어요.

면접 완료까지 약 $1 정도 소비됩니다.

3. 면접을 진행하세요.

이력서 분석이 끝나면 면접을 진행할 수 있습니다.

면접은 채팅 형식으로 진행되며 모든 질문과 답변에 대한 피드백이 구성됩니다.

면접 종료 후 각 질문에 대한 피드백과 개선사항을 확인할 수 있습니다.
